# Generalized Algebraic Theories

A GAT $\mathbb{T} = (\mathcal{S}, \mathcal{O}, \mathcal{E})$ generalizes multi-sorted equational logic with dependent sorts. Each sort $S \in \mathcal{S}$ may depend on terms of other sorts; operations $o \in \mathcal{O}$ are term constructors with typed parameters; equations $e \in \mathcal{E}$ assert judgemental equalities between terms.

panproto uses GATs as the foundation for protocol definitions. A protocol's schema theory is a GAT; a schema is a model of that theory.

## Creating a theory

```python
import panproto

theory = panproto.create_theory({
    "name": "Monoid",
    "extends": [],
    "sorts": [
        {"name": "Carrier", "params": [], "kind": "Structural"},
    ],
    "ops": [
        {"name": "mul", "params": [
            {"name": "a", "sort": "Carrier"},
            {"name": "b", "sort": "Carrier"},
        ], "output_sort": "Carrier"},
        {"name": "unit", "params": [], "output_sort": "Carrier"},
    ],
    "eqs": [],
    "directed_eqs": [],
    "policies": [],
})
print(theory)   # Theory("Monoid", sorts=1, ops=2, eqs=0)
```

## Colimits

The colimit (pushout) of theories $\mathbb{T}_1$ and $\mathbb{T}_2$ over a shared sub-theory $\mathbb{T}_0$ is:

$$\mathbb{T}_1 +_{\mathbb{T}_0} \mathbb{T}_2$$

This combines the sorts, operations, and equations of both theories, identifying the shared structure.

```python
result = panproto.colimit_theories(t1, t2, shared)
```

## Morphism checking

A theory morphism $f: \mathbb{T}_1 \to \mathbb{T}_2$ maps sorts to sorts and operations to operations, preserving all equations. `check_morphism` verifies these conditions:

```python
panproto.check_morphism(morphism_spec, domain_theory, codomain_theory)
```

Raises `GatError` if any condition fails.

## Models

A model $M$ of a theory $\mathbb{T}$ interprets each sort as a carrier set and each operation as a function on those sets. The free (initial) model is the universal model generated by all closed terms up to the equations:

```python
model = panproto.free_model(theory, max_depth=3)
violations = panproto.check_model(model, theory)
```

`check_model` returns a list of equation violation descriptions (empty if the model satisfies all equations).

## Model migration

Given a morphism $f: \mathbb{T}_1 \to \mathbb{T}_2$, the pullback functor $f^*: \mathbf{Mod}(\mathbb{T}_2) \to \mathbf{Mod}(\mathbb{T}_1)$ reindexes a model's carrier sets and operations:

```python
migrated = panproto.migrate_model(morphism_spec, model)
```
